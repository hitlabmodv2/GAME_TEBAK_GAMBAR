❓ Cara Bermain

🎯 Tujuan Game

Tebak kata atau frasa yang digambarkan dalam setiap gambar. Game ini menguji kemampuan berpikir dan kosakata Anda!

🎮 Cara Bermain

  1. Lihat gambar yang ditampilkan
  2. Ketik jawaban Anda di kotak input
  3. Klik "Jawab" untuk memeriksa jawaban
  4. Gunakan "💡 Petunjuk" jika kesulitan
  5. Klik "Lewati" jika ingin melewati soal

📊 Sistem Penilaian

  • Jawaban benar: +10 poin
  • Jawaban benar dengan petunjuk: +5 poin
  • Jawaban salah atau dilewati: 0 poin

⚙️ Fitur Lainnya

  • Tema siang/malam
  • Pilihan tingkat kesulitan
  • Efek suara (dapat dimatikan)
  • Statistik permainan
